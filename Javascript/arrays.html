Arrays in JavaScript

Array is a collection of items that are stored at contiguous memory locations.

Declare Array: 

            let arr = [5,6,7]; // An array named 'arr' with elements 5, 6 and 7

Access Elements:

To access individual element from the array we use index number.

            console.log(arr[0]); // Output: 5 (The first element)
            console.log(arr[1]); // Output: 6 (The second element)
            console.log(arr[2]); // Output: 7 (The third element)

Add Item:

We can add new items to the array using push() function.

            arr.push(8);   // Adds 8 to the end of the array
            console.log(arr); // Output: [5, 6, 7, 8]

Remove Item:

We can remove items from the array using pop() function which removes the last item from the array.

We can remove or delete items from the array using pop(), shift(), unshift() functions.
- pop(): It removes the last item from the array and returns the removed item.
- shift(): It removes the first item from the array and returns the removed item.
- unshift(): It adds new items to the beginning of the array and shifts the other items to higher indices.
- unshift(): It adds new items to the beginning of the array and returns the new length of the array.
- unshift(): It adds a new item to the beginning of the array and returns the length of the array after adding the new item.
